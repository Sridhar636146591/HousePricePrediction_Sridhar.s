# House Price Prediction

## Project Overview

This project predicts house prices using Machine Learning techniques. The goal is to analyze housing data, identify the factors that influence house prices, and build regression models to estimate property values.

## Dataset

* Dataset: Housing Prices Dataset
* Source: Kaggle
* File Used: Housing.csv

## Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

## Project Tasks

1. Data Loading and Exploration
2. Data Cleaning and Preprocessing
3. Model Building

   * Linear Regression
   * Random Forest Regressor
4. Model Evaluation

   * MAE
   * RMSE
   * R² Score
5. Data Visualization
6. Insights and Summary

## Results

The models were trained and evaluated to predict house prices. Performance was compared using standard regression metrics, and key features affecting house prices were identified through data analysis.
